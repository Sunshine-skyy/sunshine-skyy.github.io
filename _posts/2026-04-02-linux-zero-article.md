# Vim 核心概念：三种模式（小白必读）
Vim 和普通编辑器的最大区别是它拥有**模式**。理解这三种模式，你就学会了 Vim 的一半。
## 一、三种模式是什么？
| 模式 | 你能做什么 | 底部提示 |
|------|-----------|----------|
| **正常模式** (Normal) | 移动光标、删除、复制、粘贴、保存、退出 | 无提示或只显示文件名 |
| **插入模式** (Insert) | 像普通记事本一样输入文字 | `-- INSERT --` |
| **命令行模式** (Command-line) | 输入 `:` 开头的命令（保存、退出、查找等） | 冒号 `:` 出现在底部 |
## 二、如何切换模式？

按 i / a / o 等  
正常模式 ──────────→ 插入模式  
↑ │  
└── 按 Esc ────────┘

正常模式 ── 按 : ──→ 命令行模式  
↑ │  
└─—─ 按 Esc ────────┘
- **正常模式 → 插入模式**：按 `i`（光标前插入）、`a`（光标后插入）、`o`（下一行新开一行）
- **任何模式 → 正常模式**：按 `Esc` 键（这是你的“安全键”）
- **正常模式 → 命令行模式**：按 `:` 键

> 💡 **最重要的操作**：任何时候不确定自己在什么模式，多按几次 `Esc` 就能回到正常模式。
>  💡 **小tips**：无论是 i / a / o 还是其他的操作，都一定要在英文模式下按哦，否则在中文模式下输入Vim 无法识别，命令就失效了。

## 三、如何识别当前模式？

看 Vim 窗口的**底部左下角**：
- 显示 `-- INSERT --` → 你在插入模式
- 显示 `:` → 你在命令行模式（等待输入命令）
- 什么都没有或只显示文件名 → 你在正常模式

## 四、保存与退出（命令行模式）

在正常模式下按 `:` 进入命令行模式，然后输入：

| 命令 | 作用 |
|------|------|
| `:q` | 退出（未保存时会提示） |
| `:q!` | 强制退出，不保存 |
| `:w` | 保存 |
| `:wq` | 保存并退出 |

**完整操作流程**：
1. 按 `Esc` 确保在正常模式
2. 按 `:`（底部出现冒号）
3. 输入 `wq`（或 `q!`）
4. 按 `Enter`

## 五、常见误区

| 错误现象 | 原因 | 解决方法 |
|----------|------|----------|
| 按字母却在屏幕上打字 | 你在插入模式，想执行命令 | 按 `Esc` 回到正常模式 |
| 按 `:wq` 没反应，只看到冒号 | 可能处于插入模式，冒号被当成字符 | 按 `Esc`，再按 `:` |
| 不知道怎么退出 Vim | 没记住命令 | 按 `Esc`，再按 `:`，输入 `q!` 强制退出 |

## 六、练习建议

1. 打开终端，输入 `vim test.txt`
2. 观察底部（应该是正常模式，无提示）
3. 按 `i`，底部出现 `-- INSERT --`，输入几个字母
4. 按 `Esc`，`-- INSERT --` 消失，回到正常模式
5. 按 `:`，底部出现冒号，输入 `q!` 回车退出

重复几次，直到你熟练切换模式。
